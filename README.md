# Wokelo-AI-MCP
Wokelo MCP connects AI assistants to a dealmaking research platform with 100M+ companies, 30+ premium data sources, and deep M&amp;A and fundraising data. It offers 25 tools for screening, mapping, and research workflows. All data is source-verified (no hallucinations) and SOC 2 Type II &amp; ISO 27001 compliant.
